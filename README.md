# 🏦 Banking System Core

[![C#](https://img.shields.io/badge/Language-C%23%2010.0%2B-blue)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![Platform](https://img.shields.io/badge/Platform-.NET%20%2F%20Visual%20Studio-purple)](https://visualstudio.microsoft.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Надежное ядро банковской системы, реализованное на C#. Проект демонстрирует глубокое применение принципов объектно-ориентированного программирования (ООП) для управления финансовыми потоками, мультивалютными счетами и инвестиционными портфелями.

---

## 🏛 Архитектура и ООП

Проект построен на четкой иерархии классов и строгом контроле состояний, что обеспечивает безопасность финансовых операций[cite: 1]:

*   **Абстракция:** Базовый класс `AbstractAccount` задает фундамент для всех типов счетов[cite: 1].
*   **Инкапсуляция:** Вся логика валидации и изменения баланса скрыта внутри классов. Прямой доступ к критическим данным ограничен для предотвращения некорректных состояний[cite: 1].
*   **Наследование:** Специализированные счета (`Saving`, `Premium`, `Investment`) расширяют возможности базового `BankAccount`[cite: 1].
*   **Полиморфизм:** Методы `Withdraw` (снятие) и `Deposit` (пополнение) переопределены для каждого типа счета с учетом комиссий, лимитов и овердрафтов[cite: 1].

## 💼 Типы счетов и возможности

### 💳 Standard Account (`BankAccount`)
*   Автоматическая генерация уникального 8-значного ID[cite: 1].
*   **Мультивалютность:** Поддержка RUB, USD, EUR, KZT, CNY[cite: 1].
*   **Жизненный цикл:** Статусы счета (Активен, Заморожен, Закрыт)[cite: 1].

### 📈 Saving Account (`SavingAccount`)
*   **Пассивный доход:** Ежемесячное начисление процентов на остаток[cite: 1].
*   **Безопасность:** Контроль неснижаемого остатка (`MinBalance`)[cite: 1].

### 💎 Premium Account (`PremiumAccount`)
*   **Гибкость:** Настраиваемый лимит овердрафта (возможность уходить в минус)[cite: 1].
*   **Лимиты:** Установленные суточные ограничения на снятие наличных[cite: 1].
*   **Комиссии:** Фиксированная комиссия за проведение операций[cite: 1].

### 📊 Investment Account (`InvestmentAccount`)
*   **Портфель:** Управление активами (акции, облигации и др.)[cite: 1].
*   **Аналитика:** Расчет прогнозируемой годовой доходности и текущей стоимости портфеля[cite: 1].
*   Разделение основного и инвестиционного балансов[cite: 1].

## 🛡 Безопасность и Валидация

Система включает в себя многоуровневую проверку транзакций[cite: 1]:
1.  **Контроль средств:** Автоматическая проверка баланса с учетом лимитов овердрафта[cite: 1].
2.  **Статус-фильтр:** Блокировка операций для замороженных или закрытых счетов[cite: 1].
3.  **Валидация данных:** Строгая проверка имен владельцев, сумм и идентификаторов[cite: 1].

## 🚀 Как использовать

1.  Клонируйте репозиторий.
2.  Откройте файл решения `BankingSystem.sln` в **Visual Studio 2022**[cite: 1].
3.  Скомпилируйте проект (Build Solution).
4.  Используйте пространство имен `BankingSystem.Models` для интеграции[cite: 1].

**Пример кода:**
```csharp
using BankingSystem.Models;
using BankingSystem.Enums;

// Создание сберегательного счета
var mySavings = new SavingAccount("Ян", Currency.RUB, monthlyInterestRate: 1.5m, initialBalance: 50000);

// Применение месячного процента
mySavings.ApplyMonthlyInterest();
```
